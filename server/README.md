# flitter-server

# Requirements
* Node.js v6.10.2 LTS or greater
* Docker
  * postgres image
    * `docker pull postgres`

# To create the database
`npm run db:create`

# To start the database
`npm run db:start`

# To start the server
`npm start`

# To stop the server
`Ctrl+C`

# To stop the database
`npm run db:stop`

# To remove the database
`npm run db:rm`
